# Social Spark App

## Overview
The social spark app is a simple Android app that I created as part of this project using Kotlin in Android Studio. The app is designed to provide users with simple social activity suggestions based on the time of the day they enter. It focuses on being easy to use while encouraging users to stay socially active in their daily lives. 

## Purpose of the App
The purpose of this application is to encourage people to stay socially active, even on busy days. By entering a time like morning, afternoon or night, the app provides a simple and practical activity suggestion that the user easily follow. This helps users stay connected and maintain social interaction in their daily lives.

## Features
•	The user can enter a time of the day, such as morning afternoon, or evening.
•	The app provides a relevant social activity suggestion based on the user’s input.
•	A reset button allows the users to clear their input and start again.
•	The app displays an error message if the user enters an invalid or incorrect value. 

## How the App Works
The user types a time of day into the input field. When they press the “Show Suggestion” button, the app checks the input using if statements and displays a matching suggestion. If the input is empty or doesn’t match any expected values, the app shows an error message. The reset button clears both the input and the result so the user can try again.

## Example Inputs
Some examples of inputs that work in the app include:
