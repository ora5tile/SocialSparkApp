# Social Spark App

## Overview
The Social Spark App is a simple Android app I created using Kotlin in Android Studio. The main idea behind the app is to give users small social suggestions based on the time of day they enter.

## Purpose of the App
The purpose of this app is to encourage people to stay socially active, even during busy days. By entering a time of day like morning, afternoon, or night, the user receives a simple suggestion of something social they can do.

## Features
- The user can type in time of the day
- The app shows a suggestion based on the input 
- A reset button that clears everything  
- The app shows an error message if the input is incorrect  

## How the App Works
The user types a time of day into the input field. When they press the “Show Suggestion” button, the app checks the input using if statements and displays a matching suggestion. If the input is empty or doesn’t match any expected values, the app shows an error message. The reset button clears both the input and the result so the user can try again.

## Example Inputs
Some examples of inputs that work in the app include:
- morning  
- afternoon  
- dinner  
- night
